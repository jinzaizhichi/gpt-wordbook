### 分析词义

"Returned" 是 return 的过去式和过去分词，可表示回到原处、归还物品、退回商品或恢复某种状态，例如 "She returned home late."（她很晚才回家。）和 "I returned the book to the library."（我把书还给了图书馆。）

<Pronunciation word="Returned" uk="/rɪˈtɜːnd/" us="/rɪˈtɜːrnd/" client:only />

### 列举例句

1. "He returned from vacation on Sunday." (他周日度假回来了。)
2. "I returned the form by email yesterday." (我昨天通过邮件交回了表格。)
3. "The store accepted returned items within thirty days." (这家店三十天内接受退货。)

### 词根分析

"Return" 来自拉丁语 retornare（转回），由 re-（回）与 tornare（转动）构成。

### 词缀分析

"Returned" 在 return 后加 -ed；return 本身含前缀 re-。

### 文化背景

return policy（退货政策）、tax return（纳税申报）为北美常用表达；叙事中 returned home 是高频场景描写。

### 单词变形

- 原形: return
- 过去式/过去分词: returned
- 现在分词: returning
- 名词: return
- 固定搭配: "returned home" (回家), "returned goods" (退货)

### 记忆辅助

return 是“回去、归还”，returned 就是“已经回去了或还回去了”。

### 助记故事

"After years abroad, she returned to her hometown and reopened the old bakery." (出国多年后，她回到家乡，重新开起了那家老面包店。)
