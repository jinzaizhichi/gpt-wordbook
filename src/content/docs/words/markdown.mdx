### 分析词义

"Markdown"作名词有两种常见含义。其一，在商业语境中指为刺激销售而进行的**降价、减价**（与 markup「加价」相对）。其二，在技术与写作领域，**Markdown** 是一种轻量级标记语言，由 John Gruber 与 Aaron Swartz 于 2004 年推出，允许作者用易读的纯文本语法（如 `#` 标题、`**粗体**`、链接等）编写内容，再渲染为 HTML。GitHub、Notion、各类文档站点与静态站点生成器（如本项目的 Astro + Starlight）广泛采用 Markdown 作为内容格式。

### 列举例句

1. "The store announced a markdown on winter coats to clear inventory." (商店宣布冬季外套降价以清仓。)
2. "Please write the README in Markdown so it renders nicely on GitHub." (请用 Markdown 编写 README，以便在 GitHub 上美观展示。)
3. "She converted her notes from Word to Markdown for easier version control." (她把笔记从 Word 转成 Markdown，便于版本管理。)

### 词根分析

"Markdown"由动词 "mark"（标记、标价）与副词 "down"（向下）复合而成，字面意为「把价格标低」。技术领域的 Markdown 名称强调用简单符号「标记」文本结构，与 HTML 等「重量级」标记语言形成对比；其语法灵感部分来自纯文本邮件与 wiki 传统。

### 词缀分析

"Markdown"为复合词，由 "mark" + "down" 构成，无独立前缀或后缀。技术用法中常大写首字母 "Markdown" 以区别于一般「降价」含义。

### 文化背景

在商业文化中，seasonal markdown（季末降价）是零售清库存的常规手段，与 Black Friday 等促销节点紧密相关。在程序员与写作者社群中，Markdown 已成为事实上的轻量文档标准：GitHub Flavored Markdown（GFM）、CommonMark 规范及无数编辑器（Typora、Obsidian 等）推动了其普及。许多开源项目与文档站（包括本「智析单词书」）的内容即存为 `.md` / `.mdx` 文件。

### 单词变形

- 固定搭配: "seasonal markdown" (季末降价), "write in Markdown" (用 Markdown 编写), "Markdown file" (Markdown 文件), "Markdown syntax" (Markdown 语法)
- 相关形式: Markdown → Markdowns（少见复数）；动词化口语可说 "markdown the price"

### 记忆辅助

商业义：mark（标价）+ down（向下）= 把价格往下标 = 降价。技术义：用简单的 **mark** 符号写 **down** 成文档——「标记下来」就是 Markdown。

### 助记故事

"The manager ordered a markdown on slow-selling items, while the developer wrote the product docs in Markdown and pushed them to GitHub." (经理要求对滞销品降价，而开发者用 Markdown 撰写产品文档并推送到 GitHub。)
