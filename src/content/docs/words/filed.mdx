### 分析词义

整理存档（filed the documents）；正式提交（filed a tax return 报税、filed a complaint 提出投诉）；法律 filed a lawsuit 提起诉讼。

<Pronunciation word="Filed" uk="[faɪld]" us="[faɪld]" client:only />

### 列举例句

1. "She filed the receipts under 'Travel' in the shared drive." (她把收据在共享盘里归档到“差旅”文件夹。)
2. "The company filed for bankruptcy protection in March." (该公司三月申请了破产保护。)
3. "He filed a complaint with customer service about the delay." (他就延误向客服提交了投诉。)

### 词根分析

古法语 fil（线、行列），文件按“一行行”排列引申为 file；动词 file 与名词 file 同形。

### 词缀分析

file + -ed；相关 filing（归档材料）、filer（申报人/锉刀，需看语境）。

### 文化背景

法律与财经报道高频 filed charges、filed SEC documents；办公 filed away 表示收好。 nail file 的 file 为另一同源名词“锉”。

### 单词变形

- 固定搭配: "filed a lawsuit" (起诉), "filed taxes" (报税), "filed away" (收档放好)
- 名词: file, filing

### 记忆辅助

file 是“文件夹”，filed 是“已经放进文件夹/已经交上去了”。

### 助记故事

"Investors filed questions after the earnings call revealed lower margins." (财报电话会披露利润率下降后，投资者提交了问题。)
